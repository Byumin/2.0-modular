# CLAUDE.md - 검사 실시 화면 레퍼런스

> 카드형(QuestionStep) / 집중형(QuestionEditorial) 검사 실시 화면만 다룹니다.

## 파일

- `src/app/components/AssessmentPreview.tsx` — QuestionStep(카드형) + 모드 전환 래퍼
- `src/app/components/QuestionEditorial.tsx` — 집중형 (풀스크린 1문항)

## 카드형 레이아웃

```
┌─ sticky header (bg-white/95 backdrop-blur) ─────────────┐
│ [H] 검사명/응시자  │ ①인적사항 ─ ②검사실시 ─ ③완료      │
│ ▓▓▓▓▓░░░░ progress bar (h-0.5)                          │
└─────────────────────────────────────────────────────────┘

grid lg:grid-cols-[1fr_220px], max-w-5xl
┌─────────────────────────┬──────────────┐
│ 파트1 · 페이지 1/2      │ 진행 현황     │  ← 동일 높이
├─────────────────────────┤──────────────┤
│ ┌─ 카드 (rounded-2xl) ─┐│ ○ 도넛 %     │
│ │ [번호] 문항텍스트      ││ N/8 문항완료  │
│ │ [1][2][3][4][5]       ││              │
│ │ ✓ 응답완료            ││ 문항이동      │
│ └──────────────────────┘│ ◀ 문항N/8 ▶  │
│ (×4문항, PAGE_SIZE=4)   ││              │
│                         │ 빠른이동(5열)  │
│                         │ [1][2]...[8]  │
│                         │              │
│                         │ 응답방식       │
│                         │ [카드형][집중형]│
│                         │              │
│                         │ [제출하기]     │
└─────────────────────────┴──────────────┘
```

## 집중형 레이아웃

```
▓▓▓▓▓░░░ progress bar (h-1)
[H] 검사명          N/8 응답  [제출하기]

        01              ← 80~120px 워터마크
        Question 1 of 8
        문항 텍스트

        [1] [2] [3] [4] [5]
        ─────────────────
        비동의        동의

  ◀  ● ● ○ ○ ○ ○ ○ ○  ▶   ← fixed bottom-0
```

- 응답 선택 → 350ms 후 자동 이동
- 키보드: 1~5 응답, ←→ 이동

## 핵심 규칙

- 카드 아래 이전/다음 버튼 없음 — 사이드바에서만 페이지 전환
- 사이드바: sticky top-16, 220px
- 빠른 이동: 5열 grid, max-h-[88px] overflow-y-auto
- 페이지 인디케이터와 "진행 현황" 라벨이 동일 높이 시작
